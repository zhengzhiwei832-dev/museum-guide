## 3. 页面功能规格

### 3.1 首页 (HomePage)

**布局结构**:
- 英雄区域: 背景图 `hero.png` + 标题「博物馆漫步」+ 副标题
- 博物馆卡片网格: `grid grid-cols-1 gap-6`
- 即将上线区域: 灰色占位卡片

**交互规格**:
- 卡片 Framer Motion 入场动画: `initial={{ opacity: 0, y: 30 }}` → `whileInView={{ opacity: 1, y: 0 }}`
- 卡片图片懒加载: `loading="lazy"` + `decoding="async"`
- 图片加载前显示 shimmer 骨架屏
- 点击卡片跳转 → `navigate(museum.id)`
- 设置入口: 首页右上角齿轮图标，点击打开 `SettingsModal`

**状态管理**:
- `imageLoaded: Record<string, boolean>` — 图片加载状态映射

### 3.2 模式选择 (ModeSelect)

**布局结构**:
- 顶部导航: 返回按钮 + 博物馆名称
- 中心区域: 两个大卡片纵向排列
  - 「我在做攻略 ✈️」→ pre-trip
  - 「我已在现场 📍」→ on-site

**交互规格**:
- 卡片点击 active 状态: `scale(0.98)` + 阴影变化
- Framer Motion 交错入场: `transition={{ delay: index * 0.1 }}`

### 3.3 攻略模式 (PreTripGuide)

**布局结构** — Snap 吸附容器内多页:

| 页序 | 内容 | 组件 |
|------|------|------|
| 0 | 博物馆封面 + 简介 | 内联 JSX |
| 1 | 实用信息 | `InfoBlock` × N |
| 2 | 楼层导览 | 楼层卡片列表 |
| 3~N | 代表性展品 | `ExhibitCard` × N |
| N+1~M | 冷门展品 | `ExhibitCard` × N |
| M+1 | 攻略总结 | 内联 JSX |

**交互规格**:
- Snap 容器: `scroll-snap-type: y mandatory; overflow-y: auto`
- 每页: `scroll-snap-align: start; min-height: 100dvh`
- `PageIndicator` 组件绑定 `useSnapScroll` Hook
- 展品分区标题页使用渐变背景 `from-[var(--accent)]/10`

### 3.4 现场模式选择 (OnSiteGuide)

**布局结构**:
- 顶部导航: 返回按钮 + 博物馆名称
- 两个模式卡片:
  - 「自由探索 🗺️」→ explore
  - 「推荐路线 🧭」→ route

### 3.5 自由探索 (ExploreMode)

**布局结构** — Snap 吸附容器:

| 页序 | 内容 |
|------|------|
| 0 | 楼层导览 |
| 1~N | 展品卡片（代表性 + 冷门）|

**与攻略模式区别**: 跳过封面和实用信息，直接从楼层导览开始。

### 3.6 智能路线 (SmartRoute)

**两阶段流程**:

**阶段 1 — 时间选择** (`selectedRoute === null`):
- `TimeSelector` 组件展示路线选项
- 路线按时长标签分组显示
- 每个选项卡片: 路线名称 + 时长 + 标签

**阶段 2 — 路线展示** (`selectedRoute !== null`):
- 毛玻璃导航栏: 路线名称 + 返回按钮
- 路线概览信息卡片
- `RouteTimeline` 展示站点列表
- 每站: 顺序号 + 展品名 + 停留时间 + 提示

### 3.7 展品详情 (ExhibitDetail)

**布局结构**:
- 毛玻璃导航栏: 返回按钮 + 展品名称
- 封面图: `max-h-[45dvh]` 大图，点击打开 `ImageViewer`
- `GuideModeTabs`: 大众版 / 爱好者版切换
- 内容区:
  - 有深度内容 → `PopularGuideView` 或 `EnthusiastGuideView`
  - 无深度内容 → 降级展示基础描述 + 冷知识

**交互规格**:
- 封面图点击 → 打开全屏 `ImageViewer` 查看大图
- 支持手势缩放、拖拽、双击缩放、向下滑动退出

## 4. 组件规格

### 4.1 ExhibitCard

| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| exhibit | Exhibit | 是 | 展品数据 |
| museumId | string | 否 | 博物馆 ID（用于跳转详情）|

**视觉规格**:
- 图片容器: `h-[55dvh]` + 底部渐变遮罩
- 文字区域: 可滚动 `overflow-y-auto`
- 元数据展示: 艺术家、年代、位置、语音导览号
- 详情入口: 当 `museumId` 存在且展品有深度内容时显示

### 4.2 PageIndicator

| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| current | number | 是 | 当前页码（0-based）|
| total | number | 是 | 总页数 |
| labels | string[] | 否 | 页面标题列表 |
| onNavigate | (index: number) => void | 否 | 页面导航回调 |

**双模式规格**:
- **紧凑模式**: 3px 进度条 + `${current+1}/${total}` 页码徽章
- **展开模式**: 完整目录列表，当前页高亮
- **交互**: 点击切换、长按 300ms 展开、翻页后自动隐藏 1.5s、点击外部关闭
- **动画**: `cubic-bezier(0.22, 1, 0.36, 1)` 缓动

### 4.3 ImageViewer

| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| src | string | 是 | 图片 URL |
| alt | string | 是 | 图片替代文本 |
| isOpen | boolean | 是 | 是否打开 |
| onClose | () => void | 是 | 关闭回调 |

**功能规格**:
- **手势缩放**: 双指捏合缩放，范围 1x ~ 4x
- **双击缩放**: 双击切换 1x / 2x
- **滚轮缩放**: 鼠标滚轮缩放
- **拖拽平移**: 缩放状态下可拖拽查看
- **滑动退出**: 未缩放时向下滑动退出
- **关闭按钮**: 右上角 × 按钮 + 左下角「完成」按钮
- **缩放指示器**: 底部显示当前缩放百分比
- **快捷键**: ESC 关闭

### 4.4 PopularGuideView

| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| guide | ExhibitPopularGuide | 是 | 大众版导览数据 |

**8 章节结构**: hook → importance → visualScene → characters → viewingTips → hiddenFacts → challenge → oneLiner

### 4.5 EnthusiastGuideView

| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| guide | ExhibitEnthusiastGuide | 是 | 爱好者版导览数据 |

**8 章节结构**: archive → historicalContext → artistPhase → visualAnalysis → coreElements → thematicCore → controversies → hiddenDetails

## 6. 自定义 Hook

### 6.1 useSnapScroll

```typescript
function useSnapScroll(totalPages: number): {
  containerRef: React.RefObject<HTMLDivElement>;
  currentPage: number;
  scrollToPage: (index: number) => void;
}
```

**实现机制**: Scroll 事件 + `requestAnimationFrame` 节流
- 监听容器的 `scroll` 事件（passive）
- 每帧计算 `Math.round(scrollTop / clientHeight)` 得到当前页码
- 滚动到底部（`scrollTop >= maxScroll - 2`）时强制显示最后一页
- RAF 节流 `setCurrentPage` 调用，减少渲染压力

### 6.2 useNestedScroll

```typescript
function useNestedScroll(): {
  ref: React.RefObject<HTMLElement>;
  onTouchStart: (e: React.TouchEvent) => void;
  onTouchMove: (e: React.TouchEvent) => void;
  onWheel: (e: React.WheelEvent) => void;
}
```

**用途**: 处理嵌套在 `.snap-page` 内的可滚动区域
- 阻止触摸滑动在内部滚动到达边界时触发外层 snap 滚动
- 阻止滚轮事件冒泡到外层 snap 容器
- 绑定到内部可滚动元素即可

### 6.3 useInView

```typescript
function useInView<T extends HTMLElement>(margin = '-80px'): {
  ref: React.RefObject<T>;
  isInView: boolean;
}
```

**用途**: IntersectionObserver 封装
- 元素进入视口时 `isInView` 变为 `true`
- 组件卸载时自动 disconnect
- 用于首页卡片入场动画等场景

## 7. 错误处理与降级

| 场景 | 处理方式 |
|------|---------|
| 无效 museumId | 重定向到首页 |
| 无效 exhibitId | 展示「展品未找到」提示 |
| 图片加载失败 | 保持骨架屏状态 |
| 无深度内容 | 降级展示基础描述 + 冷知识 |
| 无匹配路线 | 展示全部路线列表 |
