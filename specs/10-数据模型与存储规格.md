# 10 — 数据模型与存储规格

| 项 | 值 |
|---|---|
| 模块编号 | M1-MuseumGuide |
| 模块名称 | 博物馆漫步 WebApp |
| 文档版本 | v1.0 |
| 文档状态 | Approved |

---

## 1. 概念模型

```
Museum (博物馆)
  ├── practicalInfo (实用信息)
  ├── floors: Floor[] (楼层导览)
  ├── featuredExhibits: Exhibit[] (代表展品)
  ├── hiddenGems: Exhibit[] (冷门展品)
  └── routes: Route[] (推荐路线)
        └── stops: RouteStop[] (路线站点)
              └── exhibitId → Exhibit (关联展品)

Exhibit (展品)
  ├── popularGuide?: ExhibitPopularGuide (大众版导览)
  └── enthusiastGuide?: ExhibitEnthusiastGuide (爱好者版导览)
```

## 2. 存储引擎

| 存储类型 | 引擎 | 用途 |
|---------|------|------|
| 静态数据 | TypeScript 模块（编译时内联）| 博物馆、展品、路线等全部内容 |
| 用户偏好 | localStorage | 导览版本偏好（大众版/爱好者版）|
| 图片资源 | 外部 CDN URL（Unsplash/Wikimedia）| 展品和博物馆图片 |

## 3. 实体定义

### 3.1 Museum

| 字段 | 类型 | 必填 | 约束 | 说明 |
|------|------|------|------|------|
| id | string | 是 | 唯一，kebab-case | 博物馆标识符 |
| name | string | 是 | 非空 | 博物馆中文名称 |
| englishName | string | 是 | 非空 | 博物馆英文名称 |
| city | string | 是 | 非空 | 所在城市 |
| country | string | 是 | 非空 | 所在国家 |
| coverImage | string | 是 | HTTPS URL | 封面图片 |
| description | string | 是 | 非空 | 博物馆简介 |
| practicalInfo | PracticalInfo | 是 | — | 实用信息对象 |
| floors | Floor[] | 是 | length ≥ 1 | 楼层列表 |
| featuredExhibits | Exhibit[] | 是 | length ≥ 10 | 代表性展品 |
| hiddenGems | Exhibit[] | 是 | length ≥ 3 | 冷门展品 |
| routes | Route[] | 是 | length ≥ 2 | 推荐路线 |

### 3.2 PracticalInfo

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| tickets | string | 是 | 票价信息 |
| reservation | string | 是 | 预约方式 |
| transportation | string | 是 | 交通指引 |
| tips | string[] | 是 | 注意事项列表 |
| openingHours | string | 是 | 开放时间 |
| audioGuide | string | 否 | 语音导览信息 |

### 3.3 Exhibit

| 字段 | 类型 | 必填 | 约束 | 说明 |
|------|------|------|------|------|
| id | string | 是 | 唯一（博物馆内）| 展品标识符 |
| name | string | 是 | 非空 | 展品中文名称 |
| image | string | 是 | HTTPS URL | 展品图片 |
| period | string | 是 | 非空 | 年代/时期 |
| location | string | 是 | 非空 | 展厅位置 |
| artist | string | 否 | — | 艺术家/作者 |
| description | string | 是 | 非空 | 基础描述 |
| funFact | string | 否 | — | 趣味冷知识 |
| audioGuideNumber | string | 否 | — | 语音导览编号 |
| popularGuide | ExhibitPopularGuide | 否 | — | 大众版深度内容 |
| enthusiastGuide | ExhibitEnthusiastGuide | 否 | — | 爱好者版深度内容 |

### 3.4 ExhibitPopularGuide（大众版 8 章节）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| hook | string | 是 | 吸引眼球的开场白 |
| importance | string | 是 | 为什么重要 |
| visualDescription | string | 是 | 画面描述 |
| characterDetails | string | 是 | 人物/细节解读 |
| viewingTips | string | 是 | 观看小贴士 |
| funFact | string | 是 | 冷知识彩蛋 |
| challenge | string | 是 | 互动挑战任务 |
| oneLiner | string | 是 | 一句话总结金句 |

### 3.5 ExhibitEnthusiastGuide（爱好者版 8 章节）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| archive | object | 是 | 作品档案（medium, dimensions, currentLocation, accessionNumber）|
| historicalContext | string | 是 | 历史背景长文 |
| artistPhase | string | 是 | 艺术家创作阶段分析 |
| visualAnalysis | string | 是 | 画面形式分析 |
| detailGallery | object[] | 是 | 局部图鉴列表（title, description）|
| intellectualCore | string | 是 | 思想内核 |
| interpretationDebate | object[] | 是 | 学术解读争议（school, view）|
| hiddenGems | string[] | 是 | 细节彩蛋列表 |

### 3.6 Route

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 路线标识符 |
| name | string | 是 | 路线名称 |
| duration | string | 是 | 预计时长（如「2小时」）|
| tags | string[] | 是 | 标签（如「精华」「深度」）|
| overview | string | 是 | 路线概述 |
| designPhilosophy | string | 否 | 路线设计思路 |
| stops | RouteStop[] | 是 | 站点列表，length ≥ 3 |

### 3.7 RouteStop

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| order | number | 是 | 站点顺序（1-based）|
| exhibitId | string | 是 | 关联展品 ID |
| exhibitName | string | 是 | 展品名称（冗余，便于展示）|
| duration | string | 是 | 建议停留时间 |
| tips | string | 否 | 游览提示 |

### 3.8 Floor

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 楼层标识符 |
| name | string | 是 | 楼层名称 |
| description | string | 是 | 楼层描述 |
| sections | string[] | 否 | 分区列表 |

## 4. localStorage 存储

| Key | Value 类型 | 默认值 | 说明 |
|-----|-----------|--------|------|
| `guide-preference` | `"popular"` \| `"enthusiast"` | `"popular"` | 导览版本偏好 |

**安全性**: 无 PII（个人可识别信息），无敏感数据。

## 5. 图片资源来源

| 来源 | URL 模式 | 用途 |
|------|---------|------|
| Unsplash | `https://images.unsplash.com/...` | 博物馆封面、展品图片 |
| Wikimedia | `https://upload.wikimedia.org/wikipedia/commons/thumb/...` | 部分展品高清缩略图 |

**URL 规范**:
- 所有图片 URL MUST 使用 HTTPS
- Wikimedia 缩略图 SHOULD 使用 1280px 宽度版本
- 获取脚本: `scripts/fetch-image-urls.mjs`
- URL 缓存: `scripts/image-urls.json`

## 6. 数据完整性约束

| 约束 | 说明 |
|------|------|
| 展品 ID 唯一性 | 同一博物馆内 featuredExhibits + hiddenGems 中不允许重复 ID |
| 路线站点引用完整性 | RouteStop.exhibitId MUST 在该博物馆的展品列表中存在 |
| 图片 URL 有效性 | 所有 image 字段 MUST 为可访问的 HTTPS URL |
| 深度内容配对 | 若展品有 popularGuide，SHOULD 同时有 enthusiastGuide |
