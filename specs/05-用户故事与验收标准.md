# 05 — 用户故事与验收标准

| 项 | 值 |
|---|---|
| 模块编号 | M1-MuseumGuide |
| 模块名称 | 博物馆漫步 WebApp |
| 文档版本 | v1.0 |
| 文档状态 | Approved |

---

## 用户故事清单

### 首页与导航

**US-M1-001**: 作为旅行规划者，我希望在首页看到博物馆列表，以便快速选择感兴趣的博物馆。

| AC | 验收标准 |
|----|---------|
| AC-01 | Given 用户打开应用，When 首页加载完成，Then 展示英雄区域和至少 3 个博物馆卡片 |
| AC-02 | Given 首页已加载，When 用户点击博物馆卡片，Then 路由跳转到 `#/:museumId` 模式选择页 |
| AC-03 | Given 首页已加载，When 存在未上线的博物馆，Then 显示「即将上线」占位卡片 |

---

**US-M1-002**: 作为用户，我希望选择「做攻略」或「到现场」模式，以便获得匹配场景的导览内容。

| AC | 验收标准 |
|----|---------|
| AC-01 | Given 用户进入模式选择页，When 页面加载，Then 显示博物馆名称和两个模式卡片 |
| AC-02 | Given 模式选择页，When 点击「我在做攻略」，Then 跳转到 `#/:museumId/pre-trip` |
| AC-03 | Given 模式选择页，When 点击「我已在现场」，Then 跳转到 `#/:museumId/on-site` |
| AC-04 | Given 模式选择页，When 点击返回按钮，Then 返回首页 |

---

### 提前做攻略

**US-M1-003**: 作为旅行规划者，我希望通过垂直滑动浏览博物馆完整攻略，以便高效获取所有信息。

| AC | 验收标准 |
|----|---------|
| AC-01 | Given 进入攻略模式，When 页面加载，Then 展示博物馆封面页（第 0 页）|
| AC-02 | Given 攻略页面，When 上滑，Then 页面吸附到下一页（scroll-snap-type: y mandatory）|
| AC-03 | Given 攻略页面，When 浏览到实用信息页，Then 展示票价、预约、交通、注意事项 |
| AC-04 | Given 攻略页面，When 浏览到楼层导览页，Then 展示博物馆楼层分区信息 |
| AC-05 | Given 攻略页面，When 浏览展品卡片，Then 展示展品图片（55dvh）、名称、描述、元数据 |
| AC-06 | Given 攻略页面，When 滑到末页，Then 展示攻略总结和操作建议 |

---

**US-M1-004**: 作为用户，我希望通过页面指示器了解当前位置和总页数，以便掌握浏览进度。

| AC | 验收标准 |
|----|---------|
| AC-01 | Given 多页滑动页面，When 页面加载，Then 右侧显示紧凑模式指示器（3px 进度条 + 页码徽章）|
| AC-02 | Given 紧凑模式，When 用户点击指示器，Then 展开为完整目录模式 |
| AC-03 | Given 展开模式，When 用户点击某页名称，Then 滚动到对应页面 |
| AC-04 | Given 任意模式，When 用户翻页，Then 指示器自动隐藏 1.5s |
| AC-05 | Given 紧凑模式，When 长按 300ms，Then 展开为目录模式 |

---

### 现场导览

**US-M1-005**: 作为现场游客，我希望自由浏览展品信息，以便随走随看。

| AC | 验收标准 |
|----|---------|
| AC-01 | Given 选择「自由探索」，When 页面加载，Then 从楼层导览开始展示（跳过实用信息）|
| AC-02 | Given 自由探索页面，When 浏览展品卡片，Then 突出显示展品位置信息 |
| AC-03 | Given 自由探索页面，When 上滑，Then 页面吸附到下一展品 |

---

**US-M1-006**: 作为时间有限的游客，我希望按可用时间获取推荐路线，以便高效游览。

| AC | 验收标准 |
|----|---------|
| AC-01 | Given 选择「推荐路线」，When 页面加载，Then 显示时间选择器 |
| AC-02 | Given 时间选择页，When 用户选择时长（如 2 小时），Then 展示对应路线方案 |
| AC-03 | Given 路线展示页，When 页面加载，Then 以时间轴形式展示站点列表 |
| AC-04 | Given 路线站点，When 查看站点详情，Then 显示展品名称、停留时间、游览提示 |

---

### 展品深度导览

**US-M1-007**: 作为用户，我希望查看展品的深度导览内容，以便深入了解展品。

| AC | 验收标准 |
|----|---------|
| AC-01 | Given 展品卡片有详情入口，When 点击「详细介绍」，Then 跳转到 `#/:museumId/exhibit/:exhibitId` |
| AC-02 | Given 展品详情页，When 页面加载，Then 展示毛玻璃导航栏 + 大幅封面图 + 内容区 |
| AC-03 | Given 展品有双版本内容，When 页面加载，Then 默认展示用户偏好版本 |
| AC-04 | Given 展品详情页，When 切换 Tab，Then 内容在大众版和爱好者版之间切换 |
| AC-05 | Given 展品无深度内容，When 页面加载，Then 降级展示基础描述和趣味冷知识 |

---

**US-M1-008**: 作为艺术爱好者，我希望设置默认的导览版本偏好，以便每次自动展示我喜欢的版本。

| AC | 验收标准 |
|----|---------|
| AC-01 | Given 用户首次使用，When 无偏好记录，Then 默认使用大众版 |
| AC-02 | Given 用户切换版本，When 选择爱好者版，Then 偏好存储到 localStorage |
| AC-03 | Given 用户重新打开应用，When 读取偏好，Then 自动使用上次选择的版本 |

---

### iPhone 适配与体验

**US-M1-009**: 作为 iPhone 用户，我希望应用完美适配我的设备，以便获得原生 App 级体验。

| AC | 验收标准 |
|----|---------|
| AC-01 | Given iPhone SE（375px），When 查看页面，Then 布局完整无溢出，字体 15px |
| AC-02 | Given iPhone 标准版（390px），When 查看页面，Then 布局居中适配，字体 16px |
| AC-03 | Given iPhone Pro Max（430px），When 查看页面，Then 充分利用屏幕空间，字体 16.5px |
| AC-04 | Given 有 Dynamic Island 的机型，When 查看页面，Then 顶部安全区域适配正常 |
| AC-05 | Given 有 Home Indicator 的机型，When 查看页面，Then 底部安全区域适配正常 |

---

**US-M1-010**: 作为用户，我希望横屏时收到提示，以便切回竖屏获得最佳体验。

| AC | 验收标准 |
|----|---------|
| AC-01 | Given 用户将设备横屏，When 方向变为 landscape 且高度 < 600px，Then 显示全屏横屏警告 |
| AC-02 | Given 横屏警告显示中，When 用户切回竖屏，Then 警告消失，恢复正常内容 |

---

**US-M1-011**: 作为用户，我希望将应用添加到主屏幕，以便像原生 App 一样使用。

| AC | 验收标准 |
|----|---------|
| AC-01 | Given iOS Safari，When 用户「添加到主屏幕」，Then 以 standalone 模式打开 |
| AC-02 | Given PWA 模式，When 打开应用，Then 方向锁定为竖屏 |
| AC-03 | Given PWA 模式，When 查看状态栏，Then 主题色为米白 #FAF8F5 |
