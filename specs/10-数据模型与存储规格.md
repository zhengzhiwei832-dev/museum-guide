# 10 — 数据模型与存储规格

| 项 | 值 |
|---|---|
| 模块编号 | M1-MuseumGuide |
| 模块名称 | 博物馆漫步 WebApp |
| 文档版本 | v1.1 |
| 文档状态 | Approved |

---

## 1. 概念模型

```
Museum (博物馆)
  ├── practicalInfo (实用信息)
  ├── floorMap (楼层导览)
  │     └── floors: Floor[]
  ├── highlights: Exhibit[] (代表展品)
  ├── hiddenGems: Exhibit[] (冷门展品)
  └── recommendedRoutes: Route[] (推荐路线)
        └── stops: RouteStop[] (路线站点)
              └── exhibitId → Exhibit (关联展品)

Exhibit (展品)
  ├── popularGuide?: ExhibitPopularGuide (大众版导览)
  └── enthusiastGuide?: ExhibitEnthusiastGuide (爱好者版导览)
```

## 2. 存储引擎

| 存储类型 | 引擎 | 用途 |
|---------|------|------|
| 静态数据 | TypeScript 模块（编译时内联）| 博物馆、展品、路线等全部内容 |
| 用户偏好 | localStorage | 导览版本偏好（大众版/爱好者版）|
| 图片资源 | 外部 CDN URL（Unsplash/Wikimedia）+ 本地静态资源 | 展品和博物馆图片 |

## 3. 实体定义

### 3.1 Museum

| 字段 | 类型 | 必填 | 约束 | 说明 |
|------|------|------|------|------|
| id | string | 是 | 唯一，kebab-case | 博物馆标识符 |
| name | string | 是 | 非空 | 博物馆中文名称 |
| nameEn | string | 是 | 非空 | 博物馆英文名称 |
| city | string | 是 | 非空 | 所在城市 |
| country | string | 是 | 非空 | 所在国家 |
| coverImage | string | 是 | HTTPS URL 或本地路径 | 封面图片 |
| description | string | 是 | 非空 | 博物馆简介 |
| practicalInfo | PracticalInfo | 是 | — | 实用信息对象 |
| floorMap | FloorMap | 是 | — | 楼层导览对象 |
| highlights | Exhibit[] | 是 | length ≥ 10 | 代表性展品 |
| hiddenGems | Exhibit[] | 是 | length ≥ 3 | 冷门展品 |
| recommendedRoutes | Route[] | 是 | length ≥ 2 | 推荐路线 |

### 3.2 PracticalInfo

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| ticketPrice | string | 是 | 票价信息 |
| openingHours | string | 是 | 开放时间 |
| reservation | string | 是 | 预约方式 |
| transportation | string | 是 | 交通指引 |
| tips | string[] | 是 | 注意事项列表 |

### 3.3 FloorMap

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| overview | string | 是 | 楼层导览总览描述 |
| floors | Floor[] | 是 | 楼层列表 |

### 3.4 Floor

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 楼层标识符 |
| name | string | 是 | 楼层名称 |
| description | string | 是 | 楼层描述 |
| zones | string[] | 否 | 分区列表 |

### 3.5 Exhibit

| 字段 | 类型 | 必填 | 约束 | 说明 |
|------|------|------|------|------|
| id | string | 是 | 唯一（博物馆内）| 展品标识符 |
| name | string | 是 | 非空 | 展品中文名称 |
| nameOriginal | string | 否 | — | 展品原名（外文）|
| image | string | 是 | HTTPS URL 或本地路径 | 展品图片 |
| period | string | 是 | 非空 | 年代/时期 |
| location | string | 是 | 非空 | 展厅位置 |
| artist | string | 否 | — | 艺术家/作者 |
| description | string | 是 | 非空 | 基础描述 |
| funFact | string | 否 | — | 趣味冷知识 |
| audioGuideNo | string | 否 | — | 语音导览编号 |
| popularGuide | ExhibitPopularGuide | 否 | — | 大众版深度内容 |
| enthusiastGuide | ExhibitEnthusiastGuide | 否 | — | 爱好者版深度内容 |

### 3.6 ExhibitPopularGuide（大众版 8 章节）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| hook | string | 是 | 一句话认识它（吸引眼球的开场白）|
| importance | string | 是 | 它为什么重要 |
| visualScene | string | 是 | 画面/视觉在发生什么 |
| characters | object[] | 是 | 人物/细节速查（name, label, detail）|
| viewingTips | string[] | 是 | 怎么看：专业小贴士 |
| hiddenFacts | string[] | 是 | 你可能不知道（冷知识）|
| challenge | object | 是 | 互动挑战（task, meaning）|
| oneLiner | string | 是 | 一句话读懂主题 |

### 3.7 ExhibitEnthusiastGuide（爱好者版 8 章节）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| archive | object | 是 | 作品档案（artist, year, medium, dimensions, collection, patron, genre）|
| historicalContext | string | 是 | 时代背景/哲学与社会语境 |
| artistPhase | string | 是 | 走近艺术家/创作阶段 |
| visualAnalysis | string | 是 | 画面解析/材质、构图与美学视线 |
| coreElements | object[] | 是 | 局部与图鉴（name, identity, action, symbolism）|
| thematicCore | string | 是 | 思想内核 |
| controversies | object[] | 是 | 解读争议（school, description）|
| hiddenDetails | string[] | 是 | 细节彩蛋 |

### 3.8 Route

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 路线标识符 |
| duration | string | 是 | 预计时长（如「2小时」）|
| durationLabel | string | 是 | 时长标签（如「2h」）|
| overview | string | 是 | 路线概述 |
| designThought | string | 否 | 路线设计思路 |
| stops | RouteStop[] | 是 | 站点列表，length ≥ 3 |

### 3.9 RouteStop

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| order | number | 是 | 站点顺序（1-based）|
| exhibitId | string | 是 | 关联展品 ID |
| exhibitName | string | 是 | 展品名称（冗余，便于展示）|
| stayMinutes | number | 是 | 建议停留时间（分钟）|
| tip | string | 否 | 游览提示 |

## 4. localStorage 存储

| Key | Value 类型 | 默认值 | 说明 |
|-----|-----------|--------|------|
| `museum-guide-mode` | `"popular"` \| `"enthusiast"` | `"popular"` | 导览版本偏好 |

**安全性**: 无 PII（个人可识别信息），无敏感数据。

## 5. 图片资源来源

| 来源 | URL 模式 | 用途 |
|------|---------|------|
| Unsplash | `https://images.unsplash.com/...` | 博物馆封面、展品图片 |
| Wikimedia | `https://upload.wikimedia.org/wikipedia/commons/thumb/...` | 部分展品高清缩略图 |
| 本地静态资源 | `/images/:museumId/:filename.jpg` | 部分博物馆展品图片（如故宫博物院）|

**URL 规范**:
- 所有图片 URL MUST 使用 HTTPS（外部）或相对路径（本地）
- Wikimedia 缩略图 SHOULD 使用 1280px 宽度版本
- 本地图片 SHOULD 使用 WebP/JPG 格式，单张 ≤ 500KB

## 6. 数据完整性约束

| 约束 | 说明 |
|------|------|
| 展品 ID 唯一性 | 同一博物馆内 highlights + hiddenGems 中不允许重复 ID |
| 路线站点引用完整性 | RouteStop.exhibitId MUST 在该博物馆的展品列表中存在 |
| 图片 URL 有效性 | 所有 image 字段 MUST 为可访问的 URL |
| 深度内容配对 | 若展品有 popularGuide，SHOULD 同时有 enthusiastGuide |
